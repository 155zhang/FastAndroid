# Widget Module

### 功能介绍

### 包介绍
- address       地址选择器
- animation     动画
- banner        轮播图
- button        自定义按钮
- circleview    自定义圆形头像
- date          日期选择器
- dialog        对话框
- edittext      文本输入
- file          文件选择器
- photoview     图片缩放预览
- popwindow     弹框
- progressbar   进度条
- recyclerview  列表上下拉刷新
- textview      文本控件
- toast         自定义土司效果
- toolbar       自定义toolbar(网络断开、app有更新显示)
